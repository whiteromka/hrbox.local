I ПОДГОТОВИТЕЛЬНЫЙ ЭТАП.
1. Нужно нагенерировать 10 файлов по 100 000 рандомных строк в папке web/input.
Для этого нужно запустить консольную команду: php yii file-creator/

II ОСНОВНОЙ ЭТАП.
Можно сразу открыть проект в браузере http://hrbox.local/ Система при первом открытии сама все рассчитает и выдаст
ответ в браузере. Повторных перерасчетов не будет.
ИЛИ
Можно запусить консольный скрип который пдготовит все нужные результаты, и даже при первом открытии сайта ни каких расчетов не произойдет.
Для этого нужно запустить консольную команду: php yii file-reader/
